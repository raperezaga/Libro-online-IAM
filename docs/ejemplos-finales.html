De la tabla anterior se tiene que las dos primeras componentes principales se definen como sigue.

La primera CP es:

\[ Y_1=0.4518\ X_1 + 0.4617\ X_2 + 0.4505\ X_3 + 0.4707\ X_4 + 0.3977\ X_5 \]

La primera CP es un indicador del tamaño de los pájaros.

Todos los coeficientes de la CL que definen a la primera CP \(Y_1\) son positivos, además alrededor del 73.3% de la variación en los datos está relacionada con diferencias de tamaño, es decir, la primera CP reúne las variables que determinan el tamaño de las aves.

La segunda CP es:

\[ Y_2=0.0507\ X_1 - 0.2996\ X_2 - 0.3246\ X_3 - 0.1847\ X_4 + 0.8765\ X_5 \]

Las variables \(X_2\) (extensión de las alas), \(X_3\) (longitud del pico y cabeza) y \(X_4\) (longitud del húmero), contrastan con la variable \(X_5\) (longitud de la quilla), es decir, al aumentar el primer grupo de variables \(X_5\)-disminuye y viceversa.

Con lo anterior, se podría decir que \(Y_2\)-reúne las variables que registran la forma de las aves.

El valor tan pequeño de \(X_1\) en \(Y_2\), significa que el tamaño de los pájaros afecta poco a \(Y_2\).

En este ejemplo se tiene que: \[ X_i=\frac{X_{ik}-\bar{X}_K}{S_{k}} \ \ \ , \ \ \text{ACP Normado.} \]

Evaluación de cada CP sobre cada uno de los 49-valores normalizados de las aves.

Para la primera observación: \[ x_{11}=156, \ \ x_{12}=245, \ \ x_{13}=31.6, \ \ x_{14}=28.5, \ \ y \ \ x_{15}=20.5 \]

se le resta a cada componente su media y se divide por la desviación estándar, obteniendo: \[ z_{11}=-0.542, \ \ z_{12}=0.725, \ \ z_{13}=0.177, \ \ z_{14}=0.054, \ \ y \ \ z_{15}=-0.329 \]

luego el resultado, reemplazando en la primera CP, se obtiene el valor de la primera observación en la primera CP:

\[\begin{align*} Y_1&=0.452X_1+0.462X_2+0.451X_3+0.471X_4+0.398X_5\\ &=0.452\times (-0.542)+0.462\times (0.725)+0.451\times (0.177)\\ & +0.471\times (0.054)+0.398\times (-0.329)\\ Y_1&=0.0650. \end{align*}\]

Similarmente, el valor de la primera observación en la segunda CP es:

\[\begin{align*} Y_2&=0.050X_1-0.300X_2-0.325X_3-0.185X_4+0.876X_5\\ &=0.050\times (-0.542)-0.300\times (0.725)-0.325\times (0.177)\\ & -0.185\times (0.054)+0.876\times (-0.329)\\ Y_2&=-0.6071. \end{align*}\]

De forma similar se calculan los valores de cada observación sobre cada una de las dos primeras CP.

6.16.2 Ejemplo ACP con Individuos y Variables Suplementarias

Sobre los planos factoriales se pueden proyectar elementos que no fueron utilizados en el desarrollo del ACP, estos elementos pueden ser filas o columnas. Las fórmulas de proyección son las mismas de los elementos activos en el ACP. Se puede calcular la calidad de la representación de estos elementos suplementarios.

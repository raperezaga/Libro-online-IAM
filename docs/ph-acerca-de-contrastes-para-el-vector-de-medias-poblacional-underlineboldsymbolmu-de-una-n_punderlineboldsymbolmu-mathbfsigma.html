4.7 PH Acerca de Contrastes para el vector de medias Poblacional \(\underline{\boldsymbol{\mu}}\), de una \(N_p(\underline{\boldsymbol{\mu}} \ , \ \mathbf{\Sigma} )\)

4.7.1 \(\mathbf{\Sigma}\)-desconocida

Sea \(\underline{\mathbf{x}_1},\underline{\mathbf{x}_2},\cdots,\underline{\mathbf{x}_n}\) una m.a de una población normal \(p\)-variada con vector de medias \(\underline{\boldsymbol{\mu}}\)-desconocido y matriz de varianzas-covarianzas \(\mathbf{\Sigma}\)- desconocida, ie. \(\underline{\mathbf{x}_i} \sim N_p( \underline{\boldsymbol{\mu}}\ , \ \mathbf{\Sigma})\)

Sea \(C_{r \times p}\)-una matriz de constantes. \(C\)-contiene los coeficientes para \(r\)-combinaciones lineales simultáneas de las componentes de \(\underline{\boldsymbol{\mu}}=(\mu_1,\mu_2,\ldots,\mu_p)^t\), es decir: \[ C \underline{\boldsymbol{\mu}}= \begin{bmatrix} c_{11} & c_{12} & \cdots & c_{1p}\\ c_{21} & c_{22} & \cdots & c_{2p}\\ \vdots & \vdots & \ddots & \vdots \\ c_{r1} & c_{r2} & \cdots & c_{rp}\\ \end{bmatrix}\begin{bmatrix} \mu_1 \\ \mu_2 \\ \vdots \\ \mu_p \end{bmatrix}=\begin{bmatrix} c_{11}\mu_1 + c_{12}\mu_2 + \cdots + c_{1p}\mu_p \\ c_{21}\mu_1 + c_{22}\mu_2 + \cdots + c_{2p}\mu_p \\ \\ c_{r1}\mu_1 + c_{r2}\mu_2 + \cdots + c_{rp}\mu_p \end{bmatrix}_{r \times 1} \]

Se desea contrastar las hipótesis: \[ \begin{cases} H_0 \ : \ C \underline{\boldsymbol{\mu}} = \underline{\boldsymbol{\gamma}} \\ H_0 \ : \ C \underline{\boldsymbol{\mu}} \neq \underline{\boldsymbol{\gamma}} \end{cases} \ , \ \ \ \text{con} \ \ \ \underline{\boldsymbol{\gamma}}=\begin{pmatrix} \gamma_1 \\ \gamma_2 \\ \vdots \\ \gamma_r \end{pmatrix}-\text{Vector de Constantes}. \]

Un estimador insesgado para \(C \underline{\boldsymbol{\mu}}\) es: \(C \underline{\overline{\mathbf{x}}}\), el cual tiene la siguiente distribución:

\[ C \underline{\overline{\mathbf{x}}} \sim N_r \left(C \underline{\boldsymbol{\mu}} \ , \ C \mathbf{\Sigma}_{\underline{\overline{\mathbf{x}}}} C^t \right) \ \ , \ \ \text{es decir}: \] \[ C \underline{\overline{\mathbf{x}}} \sim N_r \left(C \underline{\boldsymbol{\mu}} \ , \ \frac{1}{n} C \mathbf{\Sigma} C^t \right), \ \ \ \text{pues}:\ \ \ \mathbf{\Sigma}_{\underline{\overline{\mathbf{x}}}}=\frac{\mathbf{\Sigma}}{n}. \]

Como \(\mathbf{\Sigma}\)-es desconocida se usa la estadística de prueba: \[ T_0^2= n (C \underline{\overline{\mathbf{x}}} - \underline{\boldsymbol{\gamma}})^t [C \mathbf{S} C^t]^{-1}(C \underline{\overline{\mathbf{x}}} - \underline{\boldsymbol{\gamma}}) \sim \frac{(n-1)r}{n-r}F_{r,n-r}. \] Se rechaza \(H_0\) :    si \(T_0^2 > \frac{(n-1)r}{n-r}F_{\alpha;\ r,n-r}\).

Ejemplo 4.16 (Ejemplo-1 PH de Contrastes de Medias) Los datos contenidos en la siguiente tabla, corresponden a los pesos (en gramos) del corcho encontrado en muestras tomadas de 28 árboles cultivados en una parcela experimental. Estas muestras fueron tomadas en 4-direcciones: Norte (N), este (E), sur (S) y oeste (O). En este caso se tienen 4-variables, que corresponden a las mediciones tomadas en cada dirección. Se quiere verificar si los pesos medios de corcho son iguales en la dirección norte-sur y en la dirección este-oeste. Se asume que estas cuatro mediciones tienen una distribución normal 4-variada, con vector de medias \(\underline{\boldsymbol{\mu}}=(\mu_N , \ \mu_E ,\ \mu_S ,\ \mu_O)\) y matriz de varianzas-covarianzas \(\mathbf{\Sigma}\), ambas cantidades desconocidas. Es decir se desea probar: \(H_0 \ : \ \mu_1=\mu_3 \ \ \text{y} \ \ \mu_2=\mu_4\).

4.7.2 \(\mathbf{\Sigma}\)-es conocida

En este caso se utiliza: \[ \chi_0^2= n (C \underline{\overline{\mathbf{x}}} - \underline{\boldsymbol{\gamma}})^t [C \mathbf{\Sigma} C^t]^{-1}(C \underline{\overline{\mathbf{x}}} - \underline{\boldsymbol{\gamma}}) \sim \chi_{r}^2. \] Se rechaza \(H_0\) :    si \(\chi_0^2 > \chi_{\alpha; r}^2\).

4.7.3 \(\mathbf{\Sigma}\)-Desconocida y n-Grande

En este caso se utiliza: \[ \chi_0^2= n (C \underline{\overline{\mathbf{x}}} - \underline{\boldsymbol{\gamma}})^t [C \mathbf{S} C^t]^{-1}(C \underline{\overline{\mathbf{x}}} - \underline{\boldsymbol{\gamma}}) \sim \chi_{r}^2. \]

Se rechaza \(H_0\) :    si \(\chi_0^2 > \chi_{\alpha; r}^2\).
